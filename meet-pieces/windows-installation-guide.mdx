## Install the AppInstaller Files

Now that you have downloaded the `.appinstaller` files, it’s time to install Pieces.

<Steps>
  <Step title="Find Saved Location">
    Open your **Downloads** folder (or wherever you saved the installer) and locate the `.appinstaller` file you just downloaded (e.g., _pieces-os.appinstaller_).
  </Step>
  <Step title="Open the Installer">
    Double-click the `.appinstaller` file to launch the installation wizard.
  </Step>
</Steps>

Installing via `.appinstaller` or `.exe` may open user-prompt windows—read on for details.

### Updating PiecesOS (AppInstaller)

If you installed via `.appinstaller` (the _Recommended_ method), quit the Pieces Desktop App before updating PiecesOS. If the update fails, ensure both PiecesOS and the Desktop App are fully closed, then retry.

### System Requirements

Your Windows device must be running **Windows 10 (1809) or higher**.  
[Check your Windows version](https://docs.pieces.app/products/meet-pieces/troubleshooting/windows#checking-windows-version).

---

| **Component**      | **Minimum**                                                                    | **Recommended**                      | **Notes**                                                        |
| ------------------ | ------------------------------------------------------------------------------ | ------------------------------------ | ---------------------------------------------------------------- |
| _CPU_              | Any modern CPU                                                                 | Multi-core CPU                       | Aim for at least 4 cores.                                        |
| _RAM (Local Mode)_ | 8 GB total system RAM with 2 GB free                                           | 16 GB total system RAM or more       | When PiecesOS runs locally.                                      |
| _RAM (Cloud Mode)_ | 8 GB total system RAM with 1 GB free                                           | 16 GB total system RAM or more       | When PiecesOS runs in cloud mode.                                |
| _Disk Space_       | 2 GB minimum (1 GB for PiecesOS + 0.5–1 GB for data), with at least 4 GB free    | 8 GB (6 GB free) or more             | Leave extra space for data and updates.                          |

## On-Screen Prompts

1. **User Account Control (UAC):** If prompted, click **Yes** to allow changes.  
2. **Install Location:** Choose your install path (default is recommended), then click **Install**.  
3. **Additional Tasks:** Optionally select creating a desktop shortcut or auto-starting the app.  
4. **Ready to Install:** Click **Install**.  
5. **Completion:** Once finished, click **Finish**.

## Enterprise & Advanced Security Considerations

If `.appinstaller` isn’t supported in your environment, use the `.exe` installers or WinGet for compatibility.

## Post-Installation Tips

- The Desktop App auto-updates.  
- To manually check for updates, open the **Profile** menu and select **Check for Updates**.

![](https://storage.googleapis.com/hashnode_product_documentation_assets/meet_pieces_assets/meet_pieces/get_started/windows/windows_check_pfd_for_updates.gif)

### Uninstalling

1. Open **Settings** > **Apps**.  
2. Search for **Pieces**.  
3. Click the three dots next to **Pieces Desktop** or **PiecesOS**, then choose **Uninstall**.
