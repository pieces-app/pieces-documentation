## Install the PKG

Once you’ve downloaded the correct `.pkg` file, it’s time to run the installer.

<Steps>
  <Step title="Open the Installer">
    Double-click the `.pkg` file to launch the macOS installer.
  </Step>
  <Step title="Follow the On-Screen Prompts">
    Navigate through the introduction screen, select the install location, and enter your administrator credentials if prompted, then click `Install Software`.
  </Step>
</Steps>

### System Requirements

There are two requirements for installing Pieces on your macOS device:

1. **Compatible OS Version**—macOS 13.0 (Ventura) or higher  
2. **Compatible Installer**—Apple Silicon (ARM) or Intel

[How to check your OS version](https://docs.pieces.app/products/meet-pieces/troubleshooting/macos#checking-os-version) | [How to check CPU architecture](https://docs.pieces.app/products/meet-pieces/troubleshooting/macos#checking-cpu-type)

---

| **Component**      | **Minimum**                                                                    | **Recommended**                      | **Notes**                                                        |
| ------------------ | ------------------------------------------------------------------------------ | ------------------------------------ | ---------------------------------------------------------------- |
| _CPU_              | Any modern CPU                                                                 | Multi-core CPU                       | Aim for at least a 4-core CPU.                                  |
| _RAM (Local Mode)_ | 8 GB total system RAM with 2 GB free                                           | 16 GB total system RAM or more       | When PiecesOS runs locally.                                     |
| _RAM (Cloud Mode)_ | 8 GB total system RAM with 1 GB free                                           | 16 GB total system RAM or more       | When PiecesOS runs in cloud mode.                               |
| _Disk Space_       | 2 GB minimum (1 GB for PiecesOS + 0.5–1 GB for data), with at least 4 GB free   | 8 GB (6 GB free) or more             | Leave extra space for data and updates.                         |

## Install the DMG

After downloading the correct `.dmg`, it’s time to install.

<Steps>
  <Step title="Locate the Installer">
    Open your **Downloads** folder and find the `.dmg` file (e.g., `Pieces.dmg`).
  </Step>
  <Step title="Mount the DMG">
    Double-click the file to mount it.
  </Step>
  <Step title="Drag & Drop">
    Drag the app icon into your **Applications** folder.
  </Step>
  <Step title="Eject the DMG">
    In Finder, right-click the mounted image and select **Eject**.
  </Step>
</Steps>

## Post-Installation Tips

Read on for update checks and uninstall instructions.

### Updating

The Pieces Desktop App auto-updates. To manually check, click the **Profile** icon > **Check for Updates**.

![](https://storage.googleapis.com/hashnode_product_documentation_assets/meet_pieces_assets/meet_pieces/get_started/macos/macos_check_pfd_for_updates_profile_dropdown.gif)

### Uninstalling

1. Open **Finder** > **Applications**  
2. Find **Pieces** and **PiecesOS**  
3. Right-click each and choose **Move to Trash**

![](https://storage.googleapis.com/hashnode_product_documentation_assets/meet_pieces_assets/meet_pieces/get_started/macos/macos_how_to_uninstall_pfd.gif)
